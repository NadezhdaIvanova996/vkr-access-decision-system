## Описание проекта

Это веб-приложение на **Streamlit**, которое в реальном времени:
- Распознаёт лица с помощью `face_recognition`
- Определяет эмоции (временная имитация модуля Темы 1)
- Вычисляет риск с помощью XGBoost-модели
- Даёт объяснение решения через **SHAP**
- Принимает решение: **ДОПУСТИТЬ / ПРОВЕРИТЬ / ОТКАЗАТЬ**

Приложение является частью командного дипломного проекта. Сейчас используются синтетические данные для эмоций и анализа личных вещей (Темы 1 и 2), так как результаты от других участников ещё не переданы.

## Как запустить локально

```bash

git clone https://github.com/твой_логин/vkr-access-decision-system.git
cd vkr-access-decision-system

# 2. Создайте виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate    # Mac/Linux
# venv\Scripts\activate     # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите приложение
streamlit run app.py
