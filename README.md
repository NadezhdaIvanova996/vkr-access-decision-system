## Описание проекта

Веб-приложение на **Streamlit**, которое в реальном времени:
- Распознаёт лица с помощью `face_recognition`
- Определяет эмоциональное состояние посетителя
- Вычисляет вероятность риска допуска с помощью модели **XGBoost**
- Объясняет каждое решение с помощью **SHAP** (waterfall-графики)
- Принимает обоснованное решение: **ДОПУСТИТЬ / ПРОВЕРИТЬ / ОТКАЗАТЬ**

Приложение использует мультимодальный подход (биометрия + эмоции + контекст + анализ личных вещей) и является завершающим модулем большого командного проекта.
